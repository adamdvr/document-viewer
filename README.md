# Document Viewer

Просмотрщик документов с поддержкой текстовых аннотаций на Angular 21.
Тестовое задание.

## Плюсы реализации

- **Минимум зависимостей.** Нет сторонних библиотек drag-and-drop,
  state-managment, UI-китов. Всё на возможностях Angular 21.
- **Signal-first.**  Для работы с состоянием используются `signal` / `computed`,
  по всему дереву компонентов включён `OnPush`.
- **Декларативная загрузка данных** через `httpResource` —
  loading / error / resolved обрабатываются прямо в шаблоне (хотя и экспериментальная фича для тестового задания решено использовать).
- **Корректная математика zoom’а.** Координаты хранятся в
  немасштабированной системе → drag и сохранённый payload
  не зависят от текущего масштаба.
- **Маленькие и чистые компоненты** — каждый отвечает за
  одну вещь.

## Минусы

- **Состояние теряется при перезагрузке.** Аннотации хранятся
  только в памяти. Минимально исправляется через
  `localStorage` + effect, или хорошее решение — отдельный POST-эндпоинт.
- **Высота страницы до загрузки картинки.** Используется
  значение 1130px до того, как сработает `img.onload`. При медленном
  соединении возможны прыжки верстки. Можно исправить добавив данные о
  ширине/высоте в API.
- ** Нет лейзи-лоадинга страниц. Можно добавить ленивую загрузку, для начала рендерить только видимые страницы, а остальные при скролле.
